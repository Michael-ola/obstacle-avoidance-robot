# Obstacle Avoidance Robot

# About

### - C++

### - Arduino IDE

### - Embedded System

## It uses a hardware device that comprises of:

### -An ultrasonic sensor for obstacle detection

### -A LIPO battery

### -Electric motors

### -Car chassis
